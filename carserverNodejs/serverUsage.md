# Carserver Usage

## get all cars

```
http://localhost:3000/cars
```

## get all types

```
http://localhost:3000/cartypes
```

## search by licence

```
http://localhost:3000/search/bylicence?value=ABC-1
```

## search by model

```
http://localhost:3000/search/bymodel?value=MbW
```
